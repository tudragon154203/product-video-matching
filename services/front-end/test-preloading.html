Pre-loading Test

Open the browser's developer console and navigate to a page with pagination to see pre-loading logs.

Instructions:

  1. Start the Next.js development server: npm run dev
  2. Visit: http://localhost:3001/en/jobs/test-job-id
  3. Open browser developer console
  4. Click the pagination next/prev buttons
  5. Look for console logs showing:
    • "Loading data for offset X, cached: true/false"
    • "Pre-loading pages for offset X: [array]"
    • "Starting pre-load for offset X"
    • "Cached page for offset X, cache size: Y"
    • "Pre-loading completed for offset X"

Expected Behavior:

  • First page load: Shows "Fetching data for offset 0"
  • After first load: Should see pre-loading for offset 10
  • Clicking next (to page 2): Should show "Using cached data for offset 10" if pre-loading worked
  • From page 2: Should pre-load offset 0 and 20
  • Going back to page 1: Should show "Using cached data for offset 0"
